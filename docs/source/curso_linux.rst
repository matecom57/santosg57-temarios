Linux
=====

Linux es un sistema operativo de código abierto (open source) basado en Unix, conocido por ser gratuito, seguro, estable y altamente personalizable. Actúa como el 
núcleo (kernel) que conecta el software con el hardware, siendo fundamental en servidores, supercomputadoras y dispositivos embebidos. A través de distribuciones 
(distros) como Ubuntu, Debian o Fedora, ofrece opciones adaptadas a usuarios novatos y avanzados. 

Este video explica la diferencia entre el kernel Linux y un sistema operativo completo:

https://youtu.be/KXI6a2Mt46o

Características Principales de Linux:
Código Abierto: Cualquier persona puede ver, modificar y distribuir el código.
Seguridad y Estabilidad: Es menos propenso a virus y fallos en comparación con otros sistemas operativos.
Distribuciones (Distros): Son versiones empaquetadas del núcleo Linux junto con herramientas de software (ej. Ubuntu, Fedora, Arch).
Versatilidad: Se ejecuta en todo tipo de hardware, desde servidores de alta potencia hasta computadoras antiguas con pocos recursos.

Servidores Web: Ejecuta la gran mayoría de los servidores de internet.

Supercomputación: El 100% de las 500 supercomputadoras más rápidas del mundo utilizan Linux.
Escritorio: Alternativa personalizable a Windows/macOS con entornos gráficos como GNOME o KDE.
